## يعمل مع ERPNext الحالي لديك

Arkan Theme لا يستبدل ERPNext — بل **يُوسّعه**. ثبّته بجانب إعدادك الحالي:

```bash
bench get-app https://github.com/sarapil/arkan_theme
bench --site your-site install-app arkan_theme
```

كل بياناتك وسير عملك وتخصيصاتك الحالية تبقى كما هي.
